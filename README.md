# gulp-boilerplate
My Gulp boilerplate 

## Plugins:
- browser-sync
- gulp-babel
- @babel-core
- @babel/preset-env
- gulp-concat
- gulp-uglify
- gulp-rename
- del
- gulp-postcss
- postcss
- gulp-sass
- autoprefixer
- cssnano

## Installing dev dependiencies :
npm i -D gulp browser-sync gulp-babel gulp-concat gulp-uglify gulp-rename del gulp-postcss postcss gulp-sass autoprefixer cssnano
npm i -D  @babel-core @babel/preset-env
